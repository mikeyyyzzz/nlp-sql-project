# NLP to SQL Generator

This project converts natural language queries into SQL queries using NLP techniques.

## 🚀 Features
- Converts plain English to SQL
- Uses NLP concepts for understanding queries
- Jupyter Notebook implementation

## 🛠 Tech Stack
- Python
- NLP
- Jupyter Notebook

## 📂 Project Structure
- `nlp_sql_project.ipynb` → Main notebook

## 🎯 Goal
To simplify database querying using natural language.

## 📌 Future Improvements
- Add UI interface
- Improve accuracy
- Support multiple databases
